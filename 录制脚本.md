## 💡 录制技巧

### 推荐工具：
- **OBS Studio**（免费，功能强大）
- **Windows Game Bar**（Win + G）
- **Screen Recorder**（浏览器插件）

### 录制设置：
- **分辨率**: 1920×1080 或 1280×720
- **帧率**: 30fps
- **音频**: 清晰的解说音
- **格式**: MP4（H.264 编码）

### 注意事项：
1. ✅ 确保网站域名清晰可见
2. ✅ 鼠标移动不要太快
3. ✅ 每个操作稍作停留（1-2秒）
4. ✅ 解说语速适中
5. ✅ 避免背景噪音

## 🔧 视频压缩（如果超过 50MB）

### 使用 FFmpeg 压缩：
```bash
ffmpeg -i input.mp4 -c:v libx264 -crf 23 -c:a aac -b:a 128k output.mp4
```

### 或使用在线工具：
- https://www.freeconvert.com/video-compressor
- https://www.aconvert.com/cn/video/mp4-to-mp4/
