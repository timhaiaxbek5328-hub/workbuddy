1. 服务概述

WorkBuddy（以下简称"我们"或"服务"）是一款为 TikTok Shop 卖家提供店铺管理服务的应用程序。通过 TikTok Shop 官方 API，我们帮助用户管理商品、订单、库存等店铺运营活动。

2. 用户资格

使用本服务，您必须：

  • 年满 18 周岁或达到您所在地区的法定成年年龄
  • 拥有有效的 TikTok Shop 卖家账号
  • 遵守 TikTok Shop 平台规则和政策
  • 具备完全民事行为能力

3. TikTok API 使用条款

3.1 授权与访问

  • WorkBuddy 通过 TikTok 官方 OAuth 2.0 授权机制访问您的 TikTok Shop 数据
  • 我们仅在您明确授权的范围内访问和使用数据
  • 您可以随时在 TikTok 开发者设置中撤销对 WorkBuddy 的授权

3.2 数据使用限制

我们承诺：

  • 仅在提供服务的必要范围内使用您的 TikTok Shop 数据
  • 不会将您的数据出售给第三方
  • 不会将您的数据用于未授权的商业用途
  • 遵守 TikTok Developer Platform 的所有政策和指南

3.3 API 合规性

用户同意：

  • 不得通过 WorkBuddy 违反 TikTok Shop 平台规则
  • 不得使用 WorkBuddy 进行欺诈、虚假交易或其他违法行为
  • 不得尝试反向工程、破解或干扰 TikTok API
  • 不得超出 API 调用限制进行恶意请求

4. 服务内容

WorkBuddy 提供以下服务功能：

  1. 商品管理：创建、编辑、删除商品信息
  2. 订单处理：查看订单详情、更新物流信息
  3. 库存管理：实时库存同步、库存预警
  4. 数据分析：销售统计、利润分析
  5. 客户服务：消息管理、客户关系维护
