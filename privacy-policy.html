WorkBuddy 隐私政策

最后更新日期：2026年3月27日

重要提示：WorkBuddy（以下简称"我们"）非常重视您的隐私。本隐私政策说明了我们如何收集、使用、存储和保护您的个人信息。

1. 信息收集

1.1 我们收集的信息类型

信息类型 具体内容 收集目的
账号信息 TikTok Shop 卖家 ID、店铺名称、邮箱地址 创建和管理您的 WorkBuddy 账号
店铺数据 商品信息、订单数据、库存信息、客户信息 提供店铺管理服务
使用数据 访问日志、操作记录、设备信息 改进服务质量、安全监控
技术数据 IP 地址、浏览器类型、操作系统 技术支持、系统优化

1.2 信息收集方式

  • TikTok API 授权：通过 TikTok 官方 OAuth 2.0 授权获取您的店铺数据
  • 自动收集：通过 Cookies、日志文件自动收集技术数据
  • 用户输入：您在使用服务时主动提供的信息

2. 信息使用

2.1 我们如何使用您的信息

  1. 提供核心服务：
    • 管理您的 TikTok Shop 商品、订单和库存
    • 同步店铺数据，确保信息实时更新
    • 生成数据分析报告
  2. 改进服务质量：
    • 分析用户使用行为，优化产品功能
    • 修复错误，提升系统性能
  3. 安全保障：
    • 检测和防止欺诈行为
    • 保护账号安全
    • 遵守法律法规要求
  4. 客户支持：
    • 响应您的咨询和请求
    • 发送服务通知和更新

2.2 信息使用限制

我们承诺不会：

  • 出售您的个人信息给第三方
  • 将您的店铺数据用于未授权的商业用途
  • 在您授权范围外访问或使用 TikTok 数据
  • 与第三方共享可识别个人身份的信息（除非法律要求）

3. 信息共享与披露

3.1 我们不会共享您的信息，除非：

  • 获得您的明确同意
  • 法律要求：响应法院命令、政府要求或法律程序
  • 保护权利：保护我们或用户的合法权益、财产或安全
  • 服务提供商：与帮助我们运营服务的第三方服务商共享（如云服务提供商）

3.2 第三方服务商

我们可能会使用以下第三方服务：

  • 云服务器提供商（如 AWS、阿里云）
  • 数据分析服务
  • 邮件服务提供商

所有第三方服务商均需签署保密协议，并仅可在提供服务的必要范围内访问数据。

4. 信息存储与保护

4.1 数据存储

  • 存储位置：数据存储在中国境内的云服务器
  • 存储期限：
    • 账号信息：账号存续期间 + 注销后 6 个月
    • 店铺数据：账号存续期间，删除账号后立即删除
    • 日志数据：保留 12 个月（安全审计需要）

4.2 安全措施

我们采取以下安全措施保护您的数据：

  • 数据加密：
    • 传输加密：HTTPS / TLS 1.3
    • 存储加密：AES-256 加密算法
  • 访问控制：
    • 严格的权限管理，最小权限原则
    • 多因素身份验证（MFA）
    • 定期访问审计
  • 安全监控：
    • 7×24 小时安全监控系统
    • 入侵检测和防御系统
    • 定期安全漏洞扫描
  • 应急响应：
    • 数据泄露应急预案
    • 72 小时内通知受影响用户

5. 您的权利

5.1 访问和更正

您有权访问和更正您的个人信息，可通过以下方式：

  • 登录 WorkBuddy 账号，在"设置"页面查看和编辑
  • 联系我们的客服团队

5.2 数据导出

您可以要求导出您的个人数据副本，格式为 JSON 或 CSV。

5.3 删除权利

您可以要求删除您的个人信息：

  • 注销账号：注销账号后，我们将删除您的所有个人数据
  • 撤销授权：您可以在 TikTok 开发者设置中撤销对 WorkBuddy 的授权

5.4 限制处理

您有权要求我们限制对您个人信息的处理，例如：

  • 不再接收营销邮件
  • 停止特定数据处理活动

6. TikTok 数据处理

6.1 TikTok API 数据使用

通过 TikTok API 获取的数据仅用于：

  • 提供店铺管理服务
  • 按照您的指令处理订单、商品、库存
  • 生成数据分析报告

6.2 TikTok 政策合规

我们严格遵守 TikTok Developer Platform 政策，包括：

  • TikTok 平台政策
  • TikTok 数据使用条款
  • TikTok API 使用限制

7. Cookies 和追踪技术

7.1 Cookies 使用

我们使用 Cookies 用于：

  • 功能性：记住您的登录状态和偏好设置
  • 安全性：防止欺诈和滥用
  • 分析：了解用户如何使用我们的服务

7.2 Cookies 管理

您可以通过浏览器设置管理或禁用 Cookies，但这可能影响服务的某些功能。

8. 儿童隐私

WorkBuddy 不面向 18 岁以下的未成年人。我们不会故意收集儿童的个人信息。如果我们发现收集了儿童的信息，将立即删除。

9. 国际数据传输

如果您的数据需要跨境传输，我们将：

  • 遵守数据出境相关法律法规
  • 采取适当的安全措施
  • 获得您的明确同意（如需要）

10. 隐私政策更新

我们可能会不时更新本隐私政策。更新后的政策将在本页面发布，重大变更将通过邮件或应用内通知告知您。

继续使用服务即表示您接受更新后的隐私政策。

11. 监管机构投诉

如果您认为我们的隐私保护存在问题，您可以向相关监管机构投诉。在中国，您可以联系：

  • 国家互联网信息办公室
  • 省级网信办

12. 联系我们

如有任何关于隐私的问题或请求，请通过以下方式联系我们的数据保护官（DPO）：

  • 电子邮件：privacy@workbuddy.com
  • 邮寄地址：[您的公司地址]
  • 网站：https://www.workbuddy.com

我们将在 30 天内回复您的请求。
